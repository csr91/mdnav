# Navegacion Basica

## Arbol

- `Enter` abre archivo o expande carpeta
- `Tab` y `Shift+Tab` cambian foco
- `,` y `.` hacen scroll del preview

## Layout

Proba `Shift+1..5` para cambiar la proporcion entre navegacion y preview.
