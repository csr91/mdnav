# Mermaid Examples

Set de ejemplos oficiales de Mermaid para probar `mdnav` con varios tipos de diagramas.

## Que podes probar

- Preview liviano de Mermaid en el documento
- `Shift+M` para abrir salida terminal, HTML o web link
- Casos lineales y casos mas complejos
